HTML Content Slider

You can put virtually anything inside each slide.

Autoheight option automatically resizes the slider based on current slide height. Same trick can be done when transition is set to fade, or in image gallery when image scale mode is set to 'none'. Provided as a module and can be easily removed from build.

If required, you can disable mouse dragging and touch navigation completely or on specific elements.

Also, you can enable arrows, or use thumbnails or bullets instead of tabs.

  • CSS
  • HTML

  width: 100%;
}

  background: #eee;
}
