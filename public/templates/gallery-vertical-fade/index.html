Gallery with Fade and Vertical Thumbs

Size of thumbnails can be easily changed dynamically via CSS media queries (as is done in this demo).

Image scaling option is set to "Fit if smaller". That means that image will fit inside slider area keeping proportions only if it's larger. There are also "fill the area", "fit", and "none" scaling options.

  • CSS
  • HTML

  width: 100%;
